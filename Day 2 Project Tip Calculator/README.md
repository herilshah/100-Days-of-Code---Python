# Tip Calculator Readme

This Python code calculates the total bill amount including tip and the amount each person should pay, based on the user input of total bill amount, tip percentage, and number of people splitting the bill.

## Usage
1. The code will prompt the user to enter the total bill amount in Rs.
2. The user will then be prompted to enter the tip .
3. The user will then be prompted to enter the number of people splitting the bill.
4. The code will then calculate the tip amount, total bill amount, and the amount each person should pay.
5. The amount each person should pay will be rounded to 2 decimal places.
6. The final calculated amount will be printed to the console in the following format:
      "Each person should pay Rs. {final_amount}"

Note: The input values should be entered as numbers. If any other input is entered, the code will throw an error.

## Example Usage
Example input:
```
Welcome to the tip calculator!
What was the total bill in Rs? 1000
How much tip would you like to give? 25 ,50 or 100? 25
How many people would you like to split the bill? 2
```
Example output:
```
Each person should pay Rs. 512.5
```
